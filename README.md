# TruthOrDrinkApp
Mobile "Truth or Drink" party game app for students, built with .NET MAUI as a school assignment. Includes requirements engineering, UI wireframes, local database, and acceptance tests.

## Documentation

This repository contains both the .NET MAUI mobile app and all documents required for the school assignment.

- **Requirements** – see [`Docs/Requirements/Requirements-TruthOrDrink.xlsx`](Docs/Requirements/Requirements-TruthOrDrink.xlsx)
- **UML & design diagrams** (use case, class, activity, state) – see [`Docs/Design/`](Docs/Design/)
- **Wireframes & navigation flow** – see [`Docs/Wireframes/`](Docs/Wireframes/)
- **Database design / ERD** – see [`Docs/Database/`](Docs/Database/)
- **Acceptance test plan** – see [`Docs/Testing/AcceptanceTests-TruthOrDrink.xlsx`](Docs/Testing/AcceptanceTests-TruthOrDrink.xlsx)
- **User manual** – see [`Docs/UserGuide/UserManual-TruthOrDrink.pdf`](Docs/UserGuide/UserManual-TruthOrDrink.pdf)
- **Demo video** – see the link in [`Docs/Media/DemoVideo.md`](Docs/Media/DemoVideo.md)

_All documents are written in Dutch, while the code and class names are in English as required by the assignment._
